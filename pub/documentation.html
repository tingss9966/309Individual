Note:To create the library with the functions to let the users be able to add in more tables, just need to use init() for the ScheduleGenerator object.

If you just need to create a single schedule you can call the makeSchedule(days, month, date,year,name) with parameters of th number of days of the schedule the month day year and the name of the schedule

API

getTodayEvents() This function can get all the events that are happening today. The return type is a list of events
getEvents(day, month) This function can get all the events that are happening on a specific date. The return type is also a list of events
getInfo() Getting the information on the library:number of tables, The total number of events. The return type is an object
changeColor(color) Changing the color of the table
